How to install
==============
* Rename web.example.xml to web.xml (in src/main/java/webapp/WEB-INF)
* Fill in the blanks

How to run
==========
```bash
mvn run:jetty
```

How to test
===========
Make sure that the server is running. You also need to have either Google Chrome or Chromium installed.
```bash
mvn test
```
